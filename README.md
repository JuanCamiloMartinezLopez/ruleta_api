# Ruleta API

Este proyecto es una API para gestionar una ruleta de apuestas. Permite crear ruletas, abrirlas, realizar apuestas y cerrarlas.

## Requisitos

- Python 3.10 o superior
- pip (gestor de paquetes de Python)
